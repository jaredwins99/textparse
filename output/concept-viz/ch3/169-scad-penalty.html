SCAD Penalty

ID 169 Section 3.1 comparison / selection Smoothly Clipped Absolute Deviation
SCAD penalty:   \( p'_\lambda(|\beta|) = \lambda \left[ \mathbf{1}(|\beta| \leq \lambda) + \frac{(a\lambda - |\beta|)_+}{(a-1)\lambda}\,\mathbf{1}(|\beta| > \lambda) \right], \quad a > 2 \)   (Fan & Li 2001, typical \(a = 3.7\))

Book Quote

"One can modify the lasso penalty function so that larger coefficients are shrunken less severely." — ESL, Section 3.1

Guarantees

OracleYes — asymptotically unbiased for large coefs
SparseYes — exact zeros for small coefs
ConvexNo — non-convex penalty
ContinuousYes — solution is continuous in z

Three SCAD Regions

|β| ≤ λ: Full L1 penalty, same as Lasso
λ < |β| ≤ aλ: Reduced penalty, smoothly tapered
|β| > aλ: Constant penalty, no shrinkage (unbiased)
