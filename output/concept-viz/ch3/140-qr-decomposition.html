ID 140 Section 3.9 Implementation / Computation

QR Decomposition

\(X = QR\)

Factors X into an orthogonal matrix Q and upper triangular R — the numerically stable way to solve least squares.

What It Means

  • 1.QR is how we actually compute β̂ = (XᵀX)⁻¹Xᵀy in practice — the normal equations are a mathematical identity, QR is the algorithm.
  • 2.Q has orthonormal columns: QᵀQ = I, meaning Q is a rotation — it preserves lengths and angles.
  • 3.R is upper triangular, so solving Rβ = Qᵀy is cheap back-substitution (O(p²)), not matrix inversion.
  • 4.Forming XᵀX directly squares the condition number of X — if X is mildly ill-conditioned, XᵀX is disastrously so. QR avoids this.

Visualization

Method comparison

Naive: Compute (XᵀX)⁻¹Xᵀy directly. Forms XᵀX in O(Np²), inverts in O(p³). Numerically unstable — condition number of XᵀX is cond(X)².

Cholesky: Form XᵀX = LLᵀ, solve two triangular systems. Saves a constant vs. naive inversion, same O(Np²+p³) class. Moderately stable.

QR (Householder): Factor X = QR in O(2Np²), solve Rβ = Qᵀy. No squaring of condition number. The standard choice.

SVD: X = UDVᵀ. Gold standard stability, handles rank-deficient X, but ~3x more flops than QR.

If QR decomposition is correct, QᵀQ must equal the identity matrix I. The heatmap of QᵀQ should show 1s on the diagonal and ~0s everywhere else. We also verify that QR reconstructs X (reconstruction error near machine epsilon).

Why this matters

Orthogonality (QᵀQ = I) means Q's columns are perpendicular unit vectors — a pure rotation. This is what makes solving Rβ = Qᵀy equivalent to the full OLS problem. The off-diagonal entries here are on the order of machine epsilon (~10⁻¹⁵), which is expected floating-point rounding — not a real error.

Book Quotes

Key Properties

Q
Q is Orthogonal
QᵀQ = I. Columns are orthonormal. Q is a rotation — it maps X columns to perpendicular unit directions.
R
R is Upper Triangular
Solves Rβ = Qᵀy by back-substitution in O(p²) — far cheaper than inverting an arbitrary matrix.
+
Stable by Design
Avoids squaring condition number. If cond(X) = 100, XᵀX has cond ≈ 10,000. QR keeps cond(X).
H
Hat Matrix is Free
H = QQᵀ — the projection matrix follows immediately from Q, no inversion required.
!
Requires Full Rank
Standard QR assumes X has full column rank. For rank-deficient X, use pivoted QR or SVD.
