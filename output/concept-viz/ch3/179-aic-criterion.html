AIC Criterion

\(\text{AIC} = -2\ell(\hat{\theta}) + 2d\)   (general)  |  For Gaussian linear model:  \(\text{AIC} = \frac{1}{N\hat{\sigma}^2_\varepsilon}\Bigl[\text{RSS} + 2d\hat{\sigma}^2_\varepsilon\Bigr]\)   where \(d\) = number of parameters

\(\text{AIC}\) = Training fit + Complexity penalty
\(= \frac{\text{RSS}}{N\hat\sigma^2}\)  +  \(\frac{2d}{N}\)

Derived from Kullback-Leibler divergence between true and fitted distribution. Penalizes each additional parameter by \(2\hat\sigma^2\).
Guarantees
✓
Consistent (asymptotically)
Selects true model as n → ∞ when true model is in candidate set (under mild conditions).
~
Unbiased estimate of pred. error
Cp / AIC are approximately unbiased for in-sample prediction error under Gaussian model. May be biased in small n.
✗
Consistent for model selection (vs BIC)
AIC tends to overfit — selects larger models than the truth in finite samples. BIC is consistent for model selection.
✓
No re-fitting needed
Computed from training fit — no held-out data required, unlike cross-validation.
✓
Prediction consistent
In predictive use: selects models with good expected prediction error, balancing bias-variance.
