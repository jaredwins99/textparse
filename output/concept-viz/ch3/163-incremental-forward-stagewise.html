Incremental Forward Stagewise Regression

Concept 163 ESL 3.8 Estimation / Optimization FSε

A slow, greedy walk through coefficient space: each step nudges the single coefficient most correlated with the current residual by a tiny increment ε.

Benefits & Tradeoffs
What does this slow walk buy you? What does it cost?
Benefits
  • Implicit regularization without a penalty term. The slow, incremental updates produce sparse, shrunk coefficient paths that closely track the lasso -- without ever writing down an L1 penalty. "The forward stagewise approach... can be shown to be closely related to the lasso" (ESL 3.8.1, p. 86).
  • Monotone coefficient paths. Each coefficient moves in one direction at a time, making the solution path interpretable and stable. No wild oscillations.
  • Works in high-dimensional settings (p >> n). Only one coefficient updates per step, so memory and computation scale gently. No matrix inversion needed.
  • Natural connection to boosting. "Buhlmann and Hothorn (2008) refer to the same procedure as 'L2boost'" (ESL 3.8, p. 88). This bridges the regression and machine learning worlds.
Tradeoffs
  • Extremely slow convergence. Requires O(1/ε) iterations to approach the OLS solution. Small ε means accurate paths but thousands of steps.
  • Step size ε is a hidden hyperparameter. Too large and paths are jagged, deviating from the lasso; too small and computation explodes. There is no free lunch.
  • Only approximates the lasso. For finite ε, the paths differ from the lasso solution. Exact equivalence requires the ε → 0 limit (FS0).
  • Greedy, not globally optimal. Each step is locally optimal (pick the most correlated predictor), which can lead to suboptimal early choices in correlated designs.
"Repeatedly updating (by a small amount ε) the coefficient of the variable most correlated with the current residuals." -- ESL 3.8, p. 86

Stagewise vs Forward Stepwise: How Does the Path Differ?
Forward stepwise makes big, greedy leaps. Forward stagewise takes tiny, cautious steps. The difference is dramatic.
Stepwise (dashed) makes full least-squares jumps -- a coefficient leaps to its full regression value in one step. Stagewise (solid) inches toward the same destination with tiny ε-increments. The stagewise path is smoother and more regularized, but far slower. "Forward stagewise regression is a more constrained version of forward stepwise" (ESL 3.8, p. 86).
Statistical Guarantees
Monotone shrinkage YES
Each coefficient moves monotonically in one direction until a different predictor becomes dominant. No oscillations or sign changes within a segment.
Sparse intermediate solutions YES
At any point along the path, only a subset of coefficients are nonzero. The algorithm enters predictors one at a time.
Path converges to lasso (ε → 0) YES
"As ε → 0, the coefficient profiles approximate... the lasso" (ESL 3.8.1, p. 87). The discrete walk becomes the continuous lasso path in the limit.
Convergence to OLS SLOW
Eventually converges to OLS as iterations → ∞, but requires O(1/ε) steps. Not practical for exact recovery; the value is in the regularized intermediate solutions.
Prediction consistency YES
Prediction error is controlled at early stopping points. Cross-validating the number of iterations provides regularization analogous to choosing λ in the lasso.
Fast convergence NO
By design, this algorithm is slow. That slowness IS the regularization mechanism. If you want speed, use LAR or coordinate descent for the lasso directly.
Formulas -- Reference
$$j^* = \arg\max_j \bigl|\langle \mathbf{x}_j,\, \mathbf{r} \rangle\bigr|$$

At each iteration, find the predictor most correlated with the current residual vector.

$$\hat\beta_{j^*} \leftarrow \hat\beta_{j^*} + \varepsilon \cdot \mathrm{sign}\bigl(\langle \mathbf{x}_{j^*},\, \mathbf{r} \rangle\bigr)$$

Nudge only that one coefficient by a fixed step ε in the direction of its correlation with the residual. All other coefficients remain unchanged.

$$\mathbf{r} \leftarrow \mathbf{y} - \mathbf{X}\hat{\boldsymbol\beta}$$

Recompute residuals after the update. The loop continues until residuals are negligible or a stopping criterion is met.

Algorithm 3.4 in ESL, p. 86. The step size ε is a small positive constant. As ε → 0, this becomes FS0 (Concept 164), which traces the lasso path exactly.
