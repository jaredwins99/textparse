Partial Least Squares

ID 159  ·  ESL Section 3.5 / discussed in §3.1 context  ·  Category: model / proxy
\(\hat{\mathbf{z}}_m = \mathbf{X}\hat{\phi}_m\) where \(\hat{\phi}_m = \arg\max_\phi \,\mathrm{Cov}(\mathbf{X}\phi,\, \mathbf{y})^2\) subject to \(\hat{\phi}_m \perp \hat{\phi}_1,\dots,\hat{\phi}_{m-1}\)

Book Quote

"Partial least squares seeks directions that have high variance and have high correlation with the response, in contrast to principal components regression which keys only on high variance."

— ESL §3.5

Guarantees

Unbiased estimator No
Consistent Conditional
Prediction consistent Yes (M=p)
Lower variance than OLS Yes
Orthogonal latent directions Yes
Orthogonal X-scores Yes

Key Facts

  • Uses M < p latent directions constructed from X
  • Each direction maximizes covariance with y (not just variance)
  • PCA ignores y; PLS does not
  • When M = p, PLS recovers the OLS solution
  • Deflates X after each component (NIPALS algorithm)
  • Can be seen as a form of regularized regression
