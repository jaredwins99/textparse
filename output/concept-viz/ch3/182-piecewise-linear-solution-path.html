Piecewise-Linear Solution Path

ESL §3.4.4 Efron et al. 2004 — Theorem LAR / Lasso / homotopy
\(\hat\beta(\lambda) \text{ is piecewise linear in } \lambda, \text{ with breakpoints where } \mathcal{A}(\lambda) \text{ changes}\)

Click a Breakpoint to See Which Variable Enters or Exits

Breakpoints

    Between any two consecutive breakpoints, the active set A(λ) is constant and β̂(λ) = β̂(λ_k) + (λ - λ_k) · v, where v is the fixed direction determined by X_A. Each breakpoint marks an exact change in the active set — one variable enters or exits. Click a breakpoint to highlight that segment.

    Book Quote

    "By construction the coefficients in LAR change in a piecewise linear fashion." — ESL §3.4.4, p. 74

    This is not an approximation — it is exact. The KKT conditions for the lasso reduce to a linear system on each segment. At each breakpoint, one constraint becomes active or inactive, inducing exactly one entry or exit from A(λ).
