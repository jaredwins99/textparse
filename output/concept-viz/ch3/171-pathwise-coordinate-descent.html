Concept 171 · Estimation / Optimization · Tier 4

Pathwise Coordinate Descent

ESL §3.8 Friedman, Hastie, Hofling & Tibshirani 2007

Benefits & Tradeoffs

Benefits

Each coordinate update is a closed-form soft-thresholding operation -- no inner optimization loop needed.
Warm starts from adjacent lambda values make the full path computation extremely fast: "nearby solutions need only a few CD sweeps to converge."
Scales to high-dimensional problems (large p) because each sweep is O(np) and sparse updates skip zero coefficients.
Converges to the exact global optimum -- the lasso objective is convex, so coordinate descent finds it.

Tradeoffs

Cannot be directly applied to non-separable penalties (e.g., group lasso) without modification.
Convergence speed depends on correlation structure: highly correlated predictors slow convergence.
Computes an approximate path on a grid of lambda values, unlike LAR/homotopy which compute the exact breakpoint path.
"Fix the penalty parameter lambda and optimize successively over each parameter, holding the other parameters fixed at their current values." -- ESL §3.8, p. 89

Coordinate Descent in Action

Watch the optimizer zigzag along coordinate axes toward the lasso solution. Each step updates one coefficient while holding the other fixed. The axis-aligned steps are the hallmark of coordinate descent -- it can only move parallel to one axis at a time.

Reference: Soft-Thresholding Update

\(\displaystyle \tilde\beta_j(\lambda) \;\leftarrow\; S\!\left(\sum_{i=1}^N x_{ij}\,r_{ij}^{(j)},\;\lambda\right), \qquad r_i^{(j)} = y_i - \sum_{k \neq j} x_{ik}\hat\beta_k\)
\(\displaystyle S(z, \lambda) = \operatorname{sign}(z)\,(|z| - \lambda)_+\)

The soft-thresholding operator S is the exact closed-form solution for each coordinate subproblem. With standardized predictors, one full sweep over all p parameters costs O(Np). Convergence to the global optimum is guaranteed because the lasso objective is convex and separable across coordinates when the others are held fixed.
