Key Facts

"Least angle regression uses a similar strategy, but only enters 'as much' of a predictor as it deserves."
●Requires only p steps (one per predictor) to compute full solution path
●At each step, the active predictor has equal correlation with the residual as all others in the active set
●Move in the equiangular direction — bisector of all active predictors
●Coefficients change piecewise-linearly; breakpoints mark when a new predictor enters
●Closely related to Lasso: a small modification gives the full lasso path
