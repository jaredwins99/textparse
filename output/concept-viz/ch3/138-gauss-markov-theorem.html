Principles / Guarantees Section 3.2.2 — The Gauss-Markov Theorem

Gauss-Markov Theorem

The least squares estimator has the smallest variance among all linear unbiased estimators.
What the Theorem Says (Plain English)
The Win
OLS is BLUE
Best Linear Unbiased Estimator. Among every estimator that is (1) a linear function of y, and (2) unbiased, OLS achieves the lowest possible variance.
The Scope
Only among linear, unbiased rivals
"Competitors" are other ways to compute a slope estimate using a linear combination of the y values that also happen to be unbiased. There are infinitely many — OLS beats them all.
The Catch
Does NOT guarantee minimum MSE
Biased estimators like ridge regression are outside the theorem's scope. They can have lower total error (MSE) by trading a little bias for a big drop in variance.
Required Assumptions (The Theorem Only Holds When These Are Met)
✓
Linear Model
\( y = X\beta + \varepsilon \)
The true relationship between predictors and response is linear in the parameters.
If violated: OLS is still computable but the "true" coefficients may not exist in the linear sense.
✓
Zero-Mean Errors (Unbiasedness)
\( E[\varepsilon] = 0 \)
Errors average to zero — no systematic bias in the noise.
If violated: OLS estimates are biased. The theorem's unbiasedness guarantee collapses.
✓
Homoscedasticity & No Autocorrelation
\( \text{Var}(\varepsilon) = \sigma^2 I \)
Every error has the same variance \(\sigma^2\) and no two errors are correlated. The covariance matrix is a scalar multiple of the identity — perfectly spherical noise.
If violated (e.g. heteroscedastic errors): OLS is still unbiased but no longer BLUE. Weighted Least Squares would beat it.
✗
NOT Required: Normality of Errors
\( \varepsilon \sim \mathcal{N}(0, \sigma^2 I) \) — not needed for Gauss-Markov
Normality is only needed for exact t-tests and F-tests (confidence intervals, p-values). The BLUE property holds for any error distribution satisfying the three assumptions above.
✗
NOT Required: Correct Model Specification
Gauss-Markov doesn't say the linear model is right — only that if it is, OLS is the best linear unbiased way to estimate it. A correctly specified nonlinear model could outperform OLS entirely.
Visualization 1 — What Is a "Competitor" Estimator?

A competitor estimator is any rule of the form \(\tilde{\theta} = c^\top y\) that is linear in the data y and unbiased (meaning \(E[\tilde{\theta}] = \beta\) for any true \(\beta\)). There are infinitely many such rules — the question is which has the smallest variance. Below, three estimators fit a regression line to the same dataset. Regenerate to see the pattern persist.

OLS (Gauss-Markov winner)
Linear ✓ Unbiased ✓
\(\hat{\beta} = (X^\top X)^{-1} X^\top y\)
Weights each point by how far its x is from the mean. Optimal allocation.

Rival Linear Estimator
Linear ✓ Unbiased ✓
\(\tilde{\beta} = \frac{\sum w_i (y_i - \bar{y}_w)}{\sum w_i (x_i - \bar{x}_w)}\)
Uses \(w_i \propto 1 + 0.8\,u_i\) (random unequal weights, renormalized to be unbiased). Legal competitor — but wasteful with information.

Ridge Regression
Linear ✓ Unbiased ✗
\(\hat{\beta}^{\text{ridge}} = (X^\top X + \lambda I)^{-1} X^\top y\)
Shrinks estimates toward zero. Biased — Gauss-Markov doesn't apply. Can still beat OLS on MSE.
Var: —  |  MSE: —
Each line was fit to the same 20 data points. OLS and the rival are both unbiased (centered on true β), but OLS has lower variance — its line is more stable across samples. Ridge is biased (it undershoots) but even more stable.
Visualization 2 — Sampling Distributions: Run 200 Datasets

To see variance you must imagine the estimator applied to many different datasets drawn from the same true model. Each dataset gives a slightly different slope estimate. The spread of those estimates across datasets is the variance. Run the simulation to see all three estimators' sampling distributions at once.

What to look for: All three distributions are shown. OLS (blue) and the rival (red) should be centered near the true β = 2.0 — they're both unbiased. But OLS is narrower. Ridge (orange) is shifted left (biased toward zero) but narrowest of all. Gauss-Markov says: among the blue and red, the blue always wins. It says nothing about the orange.
Visualization 3 — The Catch: MSE = Bias² + Variance

Gauss-Markov guarantees OLS wins the variance game among unbiased estimators. But total prediction error is measured by MSE = Bias² + Variance. A biased estimator can win the total game by trading a small amount of bias for a large reduction in variance.

The key insight: Gauss-Markov guarantees OLS wins the variance game. But the total game (MSE) can be won by a biased player. This is exactly why ridge regression exists — and why "the restriction to unbiased estimates is not necessarily a wise one" (ESL, p.51).
At-a-Glance: What OLS Guarantees
BLUE
(Best Linear Unbiased)?
YES
That's literally the theorem. Best = minimum variance.
Minimum Variance
(among linear unbiased)?
YES
No other unbiased linear rule can beat it.
Minimum MSE
(overall)?
NO
Ridge, LASSO, and other biased estimators can achieve lower MSE.
Requires Normality
of Errors?
NO
Only needs \(E[\varepsilon]=0\) and \(\text{Var}(\varepsilon)=\sigma^2 I\).
Formal Statement & Book Quotes
\[ \text{Var}(a^\top \hat{\beta}^{\text{OLS}}) \;\leq\; \text{Var}(c^\top y) \quad \text{for any linear unbiased } \tilde{\theta} = c^\top y \]
"The Gauss-Markov theorem states that if we have any other linear estimator θ̃ = cᵀy that is unbiased for aᵀβ, then Var(aᵀβ̂) ≤ Var(cᵀy)."

— Hastie, Tibshirani, Friedman, Elements of Statistical Learning, p.51
"However, it may well exist a biased estimator with smaller mean squared error. Such an estimator would trade a little bias for a larger reduction in variance. Biased models are the topic of Chapter 3 and beyond. The restriction to unbiased estimates is not necessarily a wise one."

— ESL, p.51
