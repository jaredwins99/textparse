Relaxed Lasso

What this fixes vs. vanilla Lasso

Benefits

Reduces shrinkage bias. Standard Lasso shrinks all coefficients toward zero, including the large, truly important ones. By refitting on only the selected set with a weaker penalty (or no penalty), the relaxed Lasso recovers larger coefficients closer to their true values.
When lambda_2 = 0, the second stage is just OLS on the selected subset -- fully debiased estimates. This gives you Lasso's variable selection plus OLS's unbiased estimation.
Can achieve oracle properties: asymptotically performs as well as if you knew which variables were truly non-zero.

Tradeoffs

Two tuning parameters (lambda_1 for selection, lambda_2 for refitting) instead of one. Requires cross-validation over a 2D grid.
If stage 1 selects wrong variables, stage 2 inherits those mistakes. The method is only as good as the initial Lasso selection.
"One can use the lasso to select the set of non-zero predictors, and then apply the lasso again -- but only to this set -- with a less restrictive tuning parameter." -- ESL, Section 3.8.5
