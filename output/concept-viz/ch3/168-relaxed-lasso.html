Relaxed Lasso

ID 168 Section 3.1 comparison / selection Two-Stage Procedure
Stage 1: \( \hat{\mathcal{S}} = \text{supp}\!\left(\hat\beta^{\text{lasso}}(\lambda_1)\right) \)   (select non-zero variables)     Stage 2: \( \hat\beta^{\text{relaxed}} = \arg\min_{\beta_{\hat{\mathcal{S}}}} \left\{ \|\mathbf{y} - X_{\hat{\mathcal{S}}}\beta\|^2 + \lambda_2 \|\beta\|_1 \right\} \)

Two-Stage Pipeline

Stage 1
Variable Selection
Apply Lasso with tuning parameter λ₁.
Identify the support: variables with non-zero coefficients.

Input: all p predictors
Output: selected subset \(\hat{\mathcal{S}} \subseteq \{1,\ldots,p\}\)
→
Stage 2
Refit on Submodel
Apply Lasso again, only on selected variables, with tuning parameter λ₂.

Input: \(X_{\hat{\mathcal{S}}}\) (selected columns)
Output: \(\hat\beta^{\text{relaxed}}\)

Book Quote

"One can use the lasso to select the set of non-zero predictors, and then apply the lasso again — but only to this set — with a less restrictive tuning parameter." — ESL, Section 3.1

Special Cases

λ₂ = 0: OLS on selected variables (debiased)
λ₁ = λ₂: reduces to standard Lasso
λ₂ < λ₁: typical use — relax regularization after selection
