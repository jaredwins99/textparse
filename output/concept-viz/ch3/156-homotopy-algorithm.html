Homotopy Algorithm

ESL §3.4.4 Osborne et al. 2000; Efron et al. 2004 Piecewise-linear lasso path
\(\hat\beta(\lambda) = \arg\min_\beta \tfrac{1}{2}\|y-X\beta\|^2 + \lambda\|\beta\|_1, \quad \lambda \downarrow 0\)

Piecewise-Linear Lasso Path with Active Set Breakpoints

#λ|Active|Event

Between consecutive breakpoints the active set \(\mathcal{A}(\lambda)\) is constant and \(\hat\beta(\lambda)\) is a linear function of \(\lambda\). At each breakpoint one predictor enters or exits the active set, and a new linear segment begins.

The homotopy algorithm starts at \(\lambda_{\max} = \max_j |x_j^T y|\) where all coefficients are zero, then decreases \(\lambda\) tracking the active set. It exploits the KKT stationarity condition: \(x_j^T(y - X\hat\beta) = \lambda \cdot \text{sign}(\hat\beta_j)\) for active \(j\), and \(|x_j^T r| \le \lambda\) for inactive \(j\).

Book Quote

"Osborne et al. (2000a) also discovered a piecewise-linear path for computing the lasso, which they called a homotopy algorithm." — ESL §3.4.4, p. 74

Independently derived from the LAR connection. Both reveal that the lasso solution path is piecewise linear in \(\lambda\), allowing exact computation of all solutions simultaneously.
