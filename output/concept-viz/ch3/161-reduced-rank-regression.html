Reduced-Rank Regression

ID 161  ·  ESL Section 3.7  ·  Category: model / proxy
\(\hat{\mathbf{B}}^{RRR}(M) = \hat{\mathbf{B}}^{OLS}\mathbf{P}_M^Y\) where \(\mathbf{P}_M^Y\) projects onto the first M canonical variates of Y given X

\(\hat{\mathbf{B}}^{RRR}(M) = \mathbf{U}_M\mathbf{U}_M^T\hat{\mathbf{B}}^{OLS}\)     rank(\(\hat{\mathbf{B}}\)) \(\leq M\)

Coefficient matrix B under rank constraint — use slider to change M

3

Training error vs Test error by rank M

Training error always decreases with M. Test error has a minimum at the true rank. Choosing M too large overfits; too small underfits.

Book Quote

"Reduced-rank regression borrows strength among responses by truncating the CCA. The regression surface is constrained to have rank M in the response space."

— ESL §3.7

Guarantees

Unbiased (any M) No
Unbiased at M = min(p,q) Yes
Lower MSE than OLS True rank M
Rank constraint hard Yes
Borrows strength across y Yes
Consistent If M correct

Key Facts

  • Solves: \(\min_B \|Y-XB\|_F^2\) s.t. rank(B) ≤ M
  • Solution uses CCA between X and Y
  • M = 0: predict all responses at mean
  • M = min(p,q): full OLS solution
  • Hard truncation (vs curds & whey smooth shrinkage)
  • Works best when true B is low-rank
