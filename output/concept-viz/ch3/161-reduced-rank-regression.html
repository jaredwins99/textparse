Reduced-Rank Regression

Benefits

  • Borrows strength across responses by forcing the coefficient matrix to be low-rank -- responses share a common regression subspace
    "Reduced-rank regression borrows strength among responses by truncating the CCA." -- ESL §3.7
  • Reduces effective number of parameters from p*q to M*(p+q-M), which can dramatically reduce variance
  • At M = min(p,q), recovers the full OLS solution -- so it always nests the unrestricted model
  • Solution has a clean closed form via CCA: just project OLS onto the first M canonical directions

Tradeoffs

  • Hard truncation: canonical directions are either fully kept or fully zeroed. No middle ground.
    "The regression surface is constrained to have rank M in the response space." -- ESL §3.7
  • Biased for any M < min(p,q) -- you are zeroing out real (if weak) signal
  • Choosing M is a model selection problem with no free lunch -- wrong M can hurt more than OLS
  • Requires CCA, which itself needs n > p + q

Training vs Test Error by Rank M

Training error always decreases with M (more parameters = better fit). Test error has a minimum near the true rank. Choosing M too large overfits; too small underfits. This is the rank selection problem.

Formula Reference
\(\displaystyle \hat{\mathbf{B}}^{\text{RRR}}(M) = \hat{\mathbf{B}}^{\text{OLS}}\,\mathbf{P}_M^Y\)

where \(\mathbf{P}_M^Y\) projects onto the first M canonical variates of Y given X.

\(\displaystyle \min_B \|\mathbf{Y} - \mathbf{X}\mathbf{B}\|_F^2 \quad \text{s.t.}\quad \mathrm{rank}(\mathbf{B}) \leq M\)

M = 0: predict all responses at their mean. M = min(p,q): full OLS. The solution lives on a spectrum from total pooling to no pooling.

Book Quote
"Reduced-rank regression borrows strength among responses by truncating the CCA. The regression surface is constrained to have rank M in the response space."
-- ESL §3.7
Key Facts
  • Solution uses CCA between X and Y
  • Hard truncation: keep M directions, drop the rest
  • Contrast with Curds & Whey: smooth shrinkage instead of hard cutoff
  • Works best when the true B is genuinely low-rank
  • Related to PCA regression but in the joint X-Y space
