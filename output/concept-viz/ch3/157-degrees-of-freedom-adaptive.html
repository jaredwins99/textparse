Degrees of Freedom for Adaptive Fit

\( \text{df}(\hat{y}) = \frac{1}{\sigma^2} \sum_{i=1}^N \text{Cov}(\hat{y}_i, y_i) \)
Effective df vs Tuning Parameter — Ridge, Lasso, Best Subset
Ridge (λ):
\( \text{df}(\lambda) = \text{tr}[X(X^\top X + \lambda I)^{-1}X^\top] = \sum_j \frac{d_j^2}{d_j^2 + \lambda} \)
OLS: df = p (number of parameters). Ridge: df(λ) ∈ [0, p]. Lasso: df ≈ number of non-zero coefficients (active set size).

Ridge: df(λ) = Σ d²ⱼ/(d²ⱼ+λ) — continuously shrinks from p (λ=0) to 0 (λ→∞). Computed from SVD singular values dⱼ. Lasso: df ≈ number of nonzero coefficients (active set size). Discrete jumps as coefficients enter/leave. Best subset: df = k (number of selected variables) but actual effective df is higher due to selection — ignored here. The dashed line shows OLS df = p.
Book Quotes
"df(ŷ) = (1/σ²) Σᵢ Cov(ŷᵢ, yᵢ). The harder that we fit to the data, the larger this covariance will be." — ESL §3.1
"For a linear fitting method, df(ŷ) = tr(H) where H is the hat matrix. For ridge regression, df(λ) = tr[X(XᵀX + λI)⁻¹Xᵀ]." — ESL §3.4
Statistical Properties
YES Generalizes df beyond linear models
YES Used in AIC/BIC: AIC = RSS/σ² + 2·df
NOTE Lasso df ≈ |active set| (Zou et al. 2007)
COND Subset selection: df > k (optimism)
KEY Links regularization to model complexity
Stein's lemma links df to the divergence of the fitted values w.r.t. the observed response. This gives a distribution-free way to estimate degrees of freedom for any smooth fitting procedure.
