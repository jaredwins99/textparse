The LAR algorithm builds the lasso path by adding one constraint at a time. The key modification: when a lasso coefficient hits zero, remove that predictor from the active set and continue. This gives the entire lasso solution path in \(O(p^3 + np^2)\) time — the same cost as a single OLS fit.

Book Quote

"These observations lead to a simple modification of the LAR algorithm that gives the entire lasso path, which is also piecewise linear." — ESL §3.4.4, p. 74

The modification: if a non-zero coefficient hits zero during a LAR step, drop that variable from the active set and re-compute the joint least squares direction. The path remains piecewise linear.
